給製作團隊的參考說明

這是用 three.js 把 4/29 製作會議 spec 做成的活的互動參考——不是最終 AR 檔，是讓製作團隊一眼看懂「組合 / 互動 / 音效觸發」要長怎樣的對齊版本。素材為 6/2 Tripo 批次生成、天空為 6/2 HDRI。
① 分層渲染
近景 3D：可單顆互動（漂浮石）。
遠景巨石：本預覽用 HDRI 全景天空直接撐密度（即建議的 Skybox 360）。
示範會議結論「少種模型 + 換色」：10 顆 hero clone + 調色鋪成石陣。
② 點擊互動
無重力推位移 + 自轉 + 縮放脈衝 + 發光，對齊「太空中被輕推」手感。
③ 聲音系統（Tone.js）
中古教堂和絃床（D Dorian，ADSR pad + organum 低音 + 人聲合唱）+ 聖詠旋律（凸顯 b7＝教堂感）+ glitch 人聲扭曲（呼應「AI 誤讀我們」雙重扭曲）；每個物件不同音色，多顆疊成 ambient。首次點擊解鎖。
④ 誤讀字幕
點擊浮現 <25 字考古誤讀，3 秒淡出。v2 可做魔戒「字逐筆燒出」。
⑤ 共鳴彩蛋
累積點擊達閾值 → 全體石陣共鳴脈衝 + 環境變色（Discover → Touch → Resonate）。
